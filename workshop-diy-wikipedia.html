Workshop-DIY is a French non-profit association (association loi 1901) dedicated to teaching science, technology, engineering, and mathematics (STEM) skills to young people through hands-on workshops involving robotics, artificial intelligence, electronics, 3D printing, programming, and cybersecurity.[1] Founded in 2023, the organization has developed an ecosystem of 39 browser-based educational applications, all free and open-source, requiring no installation.[2]

The association is notable for its approach of making complex technology accessible to children aged 8‚Äì16 through interactive web applications that connect to physical hardware such as the BBC micro:bit and ESP32 microcontrollers via Web Bluetooth.[3]

History [edit]

Workshop-DIY was founded in 2023 in France as an association loi 1901, a type of non-profit organization under French law. The founder, motivated by the observation that "children learn without realizing they're learning when the tools are fun enough," began developing browser-based applications that could interface with low-cost microcontrollers.[4]

The first application released was bit-bot, a web-based interface for controlling a Bit:Bot robot via Bluetooth Low Energy. The project expanded rapidly, reaching 39 applications by 2025, covering seven distinct educational domains.[5]

Application ecosystem [edit]

The Workshop-DIY ecosystem comprises 39 web applications, all hosted on GitHub Pages and requiring no server-side infrastructure. Applications range from robot control interfaces to AI training tools to cybersecurity sandboxes.[6]

Notable apps

bit-bot is the flagship application, enabling browser-based control of a physical robot via Bluetooth Low Energy. It features a virtual joystick, line-following mode, and obstacle avoidance, and has logged over 2,800 sessions.[7]

teachable-machine bridges browser-based AI training with physical hardware, allowing users to train gesture recognition models using Google's Teachable Machine and send classification results to a micro:bit via Web Bluetooth. It has been described as "the most technically ambitious app in the ecosystem."[8]

magic-hands uses MediaPipe hand tracking to generate real-time visual effects including confetti, neon trails, and particle explosions, triggered by hand gestures detected at 30 frames per second in-browser.[9]

ESP32 Robot Kit [edit]

The ESP32 Robot Kit Lite is the association's hardware offering, featuring an ESP32-WROOM-32 microcontroller with dual-core 240 MHz processor, 520 KB SRAM, WiFi, and Bluetooth Low Energy. The kit includes two DC motors, a servo, IR sensors for line following, and an ultrasonic sensor for obstacle detection.[10]

According to internal documentation, the robot has "survived 47 drops" from workshop tables, a figure that is described as "autobiographical."[11]

Technical architecture [edit]

All 39 applications are static single-page applications (SPAs) hosted on GitHub Pages, requiring no backend infrastructure. Key technologies include Web Bluetooth API for hardware communication, MediaPipe for computer vision, WebRTC (via PeerJS) for real-time collaboration, and xterm.js for terminal emulation.[12]

The platform offers 12 UI themes, including culturally-inspired designs named "Medina," "Alhambra," "Iznik," and "Zellige."[13]

Cultural impact [edit]

Workshop-DIY has gained recognition for its approach of embedding education within engaging, hacker-aesthetic interfaces. One parent reportedly stated: "My son now explains Docker to me. I'm a software engineer."[14] Another noted: "They learn without realizing they're learning. That's the trick."[15]

Criticism and controversies [edit]

The association has faced mild criticism for maintaining the WiFi password "password123" despite cybersecurity being a core curriculum area. The founder has described this as "tradition at this point."[16] Additionally, the official documentation lists the bug count as "42 (all features)," which has been described as "technically accurate but ethically questionable."[17]
